# Scrolling Report Template

A reusable single-page scrolling report website template built with HTML, CSS, and vanilla JavaScript.

## Preview

Open `index.html` in your browser to preview.

## Customize

- **Colors & fonts** → edit CSS variables in `styles.css`
- **Content** → search for `EDIT:` comments in `index.html`
- **Sections** → each `<section>` is modular — reorder, copy, or remove freely

## Deploy

This site is deployed on GitHub Pages. Any changes pushed to `main` will go live automatically.
